# NVIDIA Developer Programs & Resources

Andrew, these are the relevant NVIDIA programs and resources you should sign up for to maximize the value of your Jetson setup and the Bradix system.

---

## 1. NVIDIA Developer Program (Essential)
This is the foundational program for all NVIDIA developers. It provides access to the latest SDKs, developer tools, and training.

- **Status**: Free
- **Why**: Access to JetPack SDK, TensorRT, and NVIDIA Deep Learning Institute (DLI) courses.
- **Link**: [https://developer.nvidia.com/developer-program](https://developer.nvidia.com/developer-program)

---

## 2. Jetson AI Lab (Essential)
This is the primary hub for running Generative AI (LLMs, Vision Transformers) on Jetson devices. It provides optimized containers and tutorials.

- **Status**: Free / Resource Hub
- **Why**: Best practices for running Llama 3, Ollama, and other LLMs on Jetson hardware.
- **Link**: [https://www.jetson-ai-lab.com/](https://www.jetson-ai-lab.com/)

---

## 3. NVIDIA NIM (Inference Microservices)
NIM provides optimized inference microservices that can be deployed locally. It is the next evolution of AI deployment.

- **Status**: Free for Developers
- **Why**: Downloadable, high-performance microservices for LLMs and vision models.
- **Link**: [https://developer.nvidia.com/nim](https://developer.nvidia.com/nim)
- **Try APIs**: [https://build.nvidia.com/](https://build.nvidia.com/)

---

## 4. NVIDIA AI Enterprise (Trial)
A comprehensive suite of enterprise-grade AI tools and support.

- **Status**: 90-Day Free Trial
- **Why**: Access to NVIDIA Riva (Speech AI) and curated, secure containers.
- **Link**: [https://www.nvidia.com/en-us/data-center/products/ai-enterprise/](https://www.nvidia.com/en-us/data-center/products/ai-enterprise/)

---

## 5. NVIDIA Metropolis (Advocacy & Monitoring)
Metropolis is the vision AI platform for smart infrastructure.

- **Status**: Free Developer Access
- **Why**: If you decide to add smart home cameras for Cheryl's monitoring, Metropolis provides the tools for multi-camera tracking and anomaly detection.
- **Link**: [https://developer.nvidia.com/metropolis](https://developer.nvidia.com/metropolis)

---

## 6. NVIDIA Inception (Startup Program)
If Bradix evolves into a startup or a broader advocacy platform, this is the program to join.

- **Status**: Free (Application Required)
- **Why**: Cloud credits (AWS/GCP), hardware discounts, and technical support.
- **Link**: [https://www.nvidia.com/en-us/startups/](https://www.nvidia.com/en-us/startups/)

---

## 7. NVIDIA NGC Catalog
The centralized hub for all GPU-optimized software, models, and containers.

- **Status**: Free
- **Why**: Pre-trained models for medical imaging, legal document analysis, and vision AI.
- **Link**: [https://catalog.ngc.nvidia.com/](https://catalog.ngc.nvidia.com/)

---

## 8. Deep Learning Institute (DLI) Training
NVIDIA offers several self-paced courses for free to Developer Program members.

- **Status**: Many Free Courses
- **Recommended**:
  - *Getting Started with AI on Jetson Nano/Orin*
  - *Generative AI Explained*
  - *Building AI Applications with NIM*
- **Link**: [https://www.nvidia.com/en-us/training/self-paced-courses/](https://www.nvidia.com/en-us/training/self-paced-courses/)
